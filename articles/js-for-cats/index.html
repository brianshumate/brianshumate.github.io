JavaScript For Cats

A great introduction to JavaScript for new programmers is Max Ogden‘s delightful JavaScript for Cats. It is simple and uses the web browser development tools to introduce a newcomer to the language.

I have had great success in introducing my daughter to programming with JavaScript for Cats, and would recommend it to anyone who’d like to introduce someone new to programming for the web.

As a shout out to Max for this great resource (and really, all the fine open source contributions he’s made over the years) I made my own logo interpretation for the site.
