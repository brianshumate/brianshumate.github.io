Personal Projects

Here are links to some of my personal projects arranged by area of interest:

Art

  • Butts Corporation is meme not meme.
  • Is it Raining in Portland? is an art project that makes weather prediction simple and shockingly accurate; like a broken clock, it conveys valid information at least some of the time.
  • Happy Birthday is a collection of birthday wishes which use the HTML marquee tag to explore the subtle nature of memes and important days in the lives of humans.
  • There is Little Need to be Upset examines anger, popular meme culture, and ways of dealing with each.
  • Whip-o-Matic generates silly (and sometimes obscene) random messages using a vocabulary derived from Wesley Willis song lyrics.

Writing

  • Water Everywhere is a book of poetry about two bodies of water.
  • A Little Coffee Guidebook is a modest guidebook on brewing and enjoying coffee.
  • Apologies in Advance is random long form poetry written over a period from the 1990s until present time.
  • Happiness Tryouts is a collection of haiku about love, loss, loneliness, and lifetimes.
  • Ninajas and Rockstars is a collection of haiku about computers and the insane people who spend too much time with them.

Technical

Ansible Roles

I really enjoy automating all of the things with Ansible; here are some roles I made and maintain for use with it on either standalone servers or as OS X vagrant instances:

  • Ansible Couchbase Server is a role for deploying Couchbase Server clusters — includes simple tools for bucket creation and sample data loading as well.
  • Ansible Hoodie is a role for the wonderful Hoodie web application framework.
  • Ansible Hubot is a role for Hubot, the delightful chat bot.
  • Ansible Meatspace Chat is a role for Meatspace Chat version 1.
  • Ansibe Pinry is a role for the Pinry Django application.

Random Code

  • Docker Couchbase Server provides a means to generate Couchbase Server clusters in Docker containers.
  • Zendo is a minimalist Node.js command line Zendesk agent dashboard.
  • Synergipsum is a Node.js module that generates random corporate biz-speak aka synergy speak.
