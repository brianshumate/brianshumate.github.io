rover

→  October 27, 2017

I built rover as a prototype CLI project to continue learning the Go programming language.

The purpose of rover is to gather system and application data from an environment for diagnostics and troubleshooting in a vein similar to Apple’s sysdiagnose; it can then also package up the data and ship it off to an S3 bucket as well.

It is not yet nearly feature complete, but is actually somewhat usable.
